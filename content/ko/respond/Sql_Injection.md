## 대응 방법
SQL 인젝션 공격은 단 한번의 성공으로도 대량의 고객 정보, 기밀 정보가 유출될 수 있습니다.<br>
또한 다양한 우회 패턴이 가능하므로, 패턴 대응 방식으로는 전문 해커 그룹을 대응하기 매우 어렵습니다.

**PLURA는 SQL 인젝션 공격을 시도한 단 한번이라도 해당 IP 주소의 접속을 차단하므로<br>
어떠한 SQL 인젝션 공격이라도 대응 할 수 있습니다.**
